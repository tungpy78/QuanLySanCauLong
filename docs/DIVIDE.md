🖥️ 2. WEB (Admin + Staff Dashboard)

👉 Web là trung tâm điều hành

🎯 Nhóm chức năng chính
1. Quản lý sân bãi
CRUD sân (cầu lông, tennis, bóng bàn…)
Giá theo giờ / khung giờ
Trạng thái sân (đang sửa, đang dùng)
Sơ đồ sân (map layout)

👉 Cái này làm trên web vì:

nhiều dữ liệu
cần chỉnh sửa nhiều
2. Quản lý lịch đặt
Xem lịch toàn bộ sân (calendar view)
Duyệt / hủy booking
Override (ép lịch khi cần)
Phát hiện trùng lịch

👉 Web phù hợp vì:

cần nhìn tổng quan (calendar lớn)
3. Quản lý nhân viên
Tài khoản nhân viên
Phân quyền (admin, lễ tân, thu ngân)
Chấm công (nếu có)
Lịch làm việc
4. Quản lý bán hàng (POS nhẹ)
Quản lý sản phẩm (vợt, quần áo, cầu…)
Nhập hàng / tồn kho
Tạo đơn bán tại quầy
Báo cáo doanh thu

👉 Web tốt hơn vì:

nhập liệu nhanh bằng bàn phím
xử lý số liệu lớn
5. Báo cáo & thống kê
Doanh thu theo ngày/tháng
Tỷ lệ lấp sân
Sản phẩm bán chạy
Khách hàng VIP

👉 Cái này gần như bắt buộc là web

6. Cấu hình hệ thống
Giá sân theo giờ
Khuyến mãi
Chính sách hủy
Cài đặt thông báo
📱 3. APP (Customer App)

👉 App là trải nghiệm người dùng

🎯 Nhóm chức năng chính
1. Đặt sân (core feature)
Xem sân trống theo thời gian
Đặt sân nhanh (2–3 bước)
Thanh toán online (Momo, VNPay…)

👉 Đây là feature quan trọng nhất của app

2. Xem lịch cá nhân
Lịch đã đặt
Nhắc lịch (notification)
Check-in sân
3. Mua hàng
Xem sản phẩm
Đặt mua (online)
Giao hàng / nhận tại sân

👉 Không cần phức tạp như web

4. Tài khoản người dùng
Lịch sử đặt sân
Điểm tích lũy
Membership / VIP
5. Notification (rất quan trọng)
Nhắc giờ chơi
Khuyến mãi
Xác nhận booking

👉 Web không làm tốt cái này

6. QR / Check-in
Quét mã khi tới sân
Nhân viên xác nhận nhanh
⚖️ 4. Những thứ NÊN có ở cả 2 (nhưng khác level)
Chức năng	Web	App
Booking	Full control	Đặt nhanh
Sản phẩm	Quản lý	Mua
User	Quản lý	Sử dụng
Thanh toán	Kiểm soát	Thực hiện
